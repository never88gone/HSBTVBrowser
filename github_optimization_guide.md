### 什么是 Topics？
Topics 是 GitHub 的标签功能，可以帮助其他用户发现您的项目。添加相关的 Topics 可以大大提升项目的可见度。


### 如何添加 Topics？

#### 方法一：通过 GitHub 网页界面

1. 打开您的 GitHub 仓库页面：
   https://github.com/never88gone/HSBTVBrowser

2. 在仓库页面右侧，找到 "About" 部分（在 Code 按钮下方）

3. 点击 About 旁边的 ⚙️ 齿轮图标

4. 在弹出的对话框中，找到 "Topics" 输入框

5. 逐个输入上面推荐的 Topics（输入后按回车添加）：
   - apple-tv
   - tvos
   - browser
   - video-player
   - streaming
   - javascript
   - custom-scripts
   - privacy-focused
   - chinese
   - tiktok
   - douyin

6. 点击 "Save changes" 保存

#### 方法二：直接访问设置页面

访问：https://github.com/never88gone/HSBTVBrowser/settings

然后按照上面的步骤操作。
